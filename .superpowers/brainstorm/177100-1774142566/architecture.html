Core Platform Spine

Startbare local-first Spine-App mit Workspace-First-Einstieg, WorkPanel und eingebettetem Terminal

Schichten und Verantwortlichkeiten

UI / Textual

AppShell, Header, Tabbar, Statusbar, Command Palette, Slash Input, Panel Hosts

Application

WorkspaceService, SessionService, LayoutService, NavigationController, CommandDispatcher

Domain

Workspace, Session, Layout, Tab, PanelState, Command, Events, Policies

Infrastructure

SQLite store, config loader, local shell adapter, filesystem adapter, keybinding config

Runtime

PTY/process manager, stream router, background task supervisor for local terminal execution

Primary Happy Path
Launch App
Choose / Open Workspace
Resolve Layout + Session
Mount WorkPanel
Start Fresh Local Terminal in Stored CWD

Resume restores UI and workspace context, not live terminal processes.

State Boundaries
  • Persist: workspace id, cwd, layout, active tab, focused panel, panel config
  • Persist: explorer selection, panel sizes, command history
  • Do not persist live PTY internals or interactive subprocess state
  • On reopen: recreate terminal with same cwd and shell profile
